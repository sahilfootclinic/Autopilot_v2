# Real photos

Drop a photo into this folder to replace the cartoon avatar for any
person or portfolio. The file must be **named after that entry's slug**.

- Accepted formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`
- Square images look best (they are cropped to a circle/rounded square)
- The next deploy picks it up automatically — no code changes needed
